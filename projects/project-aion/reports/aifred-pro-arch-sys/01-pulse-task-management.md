## Subsystem 1.1: API Server (pulse/app.py)

**Framework**: FastAPI (title="Pulse", version="1.0.0-aion")
**Source**: `pulse/app.py`
**Database**: PostgreSQL via `asyncpg` connection pool
**Pool config**: `asyncpg.create_pool(min_size=2, max_size=10)` — `app.py:45-48`
**Launcher**: `pulse/start-pulse.sh`
**No CORS middleware** — no CORSMiddleware import or usage in app.py

```
fastapi>=0.100.0

```

**No aiosqlite. No pydantic.** Has pyyaml (for taxonomy loading).

### Launcher (pulse/start-pulse.sh)

- Sources `.env` from AIFred-Pro root
- Exports `PULSE_DB_*` and `PULSE_PORT` env vars
- Uses `pulse/.venv/bin/uvicorn` (dedicated venv inside `pulse/`)
- `--background` flag: nohup + PID file at `pulse/.pulse.pid`, logs to `.claude/logs/pulse.log`
- Default: `exec uvicorn app:app --host 0.0.0.0 --port $PULSE_PORT`

## Known Issues

1. **No CORS**: No CORSMiddleware configured. Cross-origin browser requests will fail.
2. **No authentication enforcement**: `PULSE_SERVICE_TOKEN` is read but no middleware checks it on all routes.
3. **No DELETE /tasks**: Tasks can only be closed, not deleted via API (labels can be deleted).
4. **No schema.sql file**: Schema is defined inline in `init_schema()` in app.py — no separate SQL file.
