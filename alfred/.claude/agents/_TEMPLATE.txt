### Memory System
If Memory Enabled = Yes:
- Read from `.claude/agents/memory/[agent-name]/learnings.json` at start
- Update learnings at end of each session
- Store key insights, patterns, and improvements

Memory schema:
```json
{
  "last_updated": "YYYY-MM-DD HH:MM:SS",
  "runs_completed": 0,
  "learnings": [
    {
      "date": "YYYY-MM-DD",
      "insight": "Description of what was learned",
      "context": "What led to this learning"
    }
  ],
  "patterns": [
    {
      "pattern": "Description of recurring pattern",
      "frequency": "How often seen",
      "action": "What to do when this pattern occurs"
    }
  ]
}
```

### Output Requirements

1. **Session Log** (`.claude/agent-output/sessions/YYYY-MM-DD_[agent-name]_[session-id].md`)
   - Full transcript of your work
   - Status updates as you progress
   - Errors and how you resolved them
   - Final results

2. **Results File** (`.claude/agent-output/results/[agent-name]/YYYY-MM-DD_[descriptive-name].md`)
   - Polished output of your work
   - Key findings and recommendations
   - Links to relevant files or sources
   - Next steps if applicable

3. **Summary** (return to calling context)
   - 2-3 sentence overview
   - Key outcomes
   - Link to full results

4. **Memory Update** (if enabled)
   - Update learnings.json with insights from this session
   - Note patterns or improvements discovered
